# LLM-ARG: 大規模言語モデルを用いた適応ルールの自動構築と最適化

本リポジトリは、情報処理学会論文誌に投稿された以下の論文に関する実験データおよびコード履歴を公開するものです。

## 論文タイトル: 大規模言語モデルを用いた適応ルールの自動構築と最適化

## 概要

本研究では、大規模言語モデル（LLM）を活用して自己適応システムの適応ルールを自動生成・最適化するフレームワーク「LLM-ARG」を提案しています。
このリポジトリでは、評価実験において提案手法（LLM-ARG）および被験者（人間）が生成した適応ルールの全履歴を提供します。

## リポジトリ構成
- `results/Experiment1`: 各イテレーションで生成された適応ルールのソースコードが含まれています。
  - `manual`: 被験者（人間）によって設計されたコード履歴。
  - `scenarioX`: シナリオXにおいて、提案手法によって生成されたLLMとの対話履歴。
